#### Ryu
>可以选择总源码编译或直接安装，详见[官网][3]。

下面是社会主义的简单安装方式
```shell
pip3 install ryu -i https://pypi.doubanio.com/simple
```
它下载完一个包后，会卡一段时间，不要以为是真的卡了，它是在安装其它依赖项，静静等待即可
>安装完成后，就可以查看流表了

首先，生成流表
```shell
sudo mn
...
mininet> h1 ping h2
```
其次，查看流表。you can use Ctrl + Shift + t to create another tab.
```shell
sudo ovs-ofctl dump-flows s1
```
