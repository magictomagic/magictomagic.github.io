## 并行版本
#### 利用 MPI_Scatter 和 MPI_Gather
思路：先将矩阵 A 和 C 按行分为 np 块，进程号为 id 的进程读取 A 的第 id 个分块和 B；进程号为 id 的进程求解相应的 C 的第 id 个分块。
实现：见[附件][2]。
测试：调整数据规模为 1000 * 2000 * 800，既耗时不长，又可体现并行和串行的明显差异
